How to generate Reports

Please follow these steps to generate a report

  • Select The Type of Report - e.g PAYE
  • Select Payroll Center - e.g Ugandan Nationals
  • Select the Payment Year - e.g 2017
  • Select the payment months that you want to reports to begin with - e.g January
  • Select the payment end payment month - e.g Jauary
NOTE

If you want to generate a report for one payment period, select the same Month for both Start and End Months. The variables used in the above example will generate the PAYE REPORT for Ugandan National's payroll center for the of January 2017.
